现有N件物品和一个容量为 V的背包，每件物品 i都有其重量w(i)和价值v(i)。问：如何选择物品装入背包，使得总重量不超过背包容量，且总价值最大？

想当然的一个解决方法是，将物品的价值除以重量得到的值作为优先度，然后按有限度从大到小选择，但是我们会发现，在背包有限的情况下，将背包装满有时比装下单位价值更高的背包更优，比如下面的这个例子：

背包容量4，有物品的价值与重量分别为3/2，3/2，5/3，显然最优解是拿两个3/2的刚好装满背包而不是性价比高的5/3。那么我们需要找到一个更好的方法，以应对各种情况。

我们从最简单的情况看起，再逐步推导，试想，当只有一件物品的情况下，我们建立列表dp [V]，其中V为背包容量，初始化列表值全为0。（列表名dp是Dynamic Programming的缩写，不过无关紧要）那么我们从V到0遍历（为什么倒着来后面讲）每次遍历，如果i>w(1)，即大于物品的重量，就把物品放入背包，遍历结束后，我们得到了一串数字，假设物品1价值为2，重量为3，背包大小为5，我们得到的列表为

0 1 2 3 4 5 索引（重量）

0 0 0 2 2 2 值（价值）

这个列表代表着在背包的容量分别为0 1 2 3 4 5的时候能达到的最大价值。

那么我们增加一个物品2，价值为1，重量为1，然后我们将列表扩展为二维列表，让列表的第二个维度长度等于物品个数，初始化全为0。依据上面的列表，我们再遍历一遍。

索引（容量） 0 1 2 3 4 5

值（价值） 0 0 0 2 2 2

0 0 0 0 0 0

第一步，索引（容量）为5，大于1，那么我们有两个选择：在前面的基础上再放入物品2（5也大于两个物品的重量和），以及只放入物品2。显然直接放入更优，那么列表变为：

索引（容量） 0 1 2 3 4 5

值（价值） 0 0 0 2 2 2

0 0 0 0 0 3

继续这个操作直到列表为

索引（容量） 0 1 2 3 4 5

值（价值） 0 0 0 2 2 2

0 0 0 0 3 3

当遍历到i = 3时，我们发现不能把两个物品同时放入。那么我们比较不放入第一个以及不放入第二个的情形，i = 2，1时只能放入第二个物品。列表为：

索引（容量） 0 1 2 3 4 5

值（价值） 0 0 0 2 2 2

0 1 1 2 3 3

但是我们如果单纯看“要不要放入第二个”，我们会发现，在物品多的时候选择也会变多，比如我们添加物品3，价值为3，重量为2，添加新行，初始化全为0。

索引（容量） 0 1 2 3 4 5

值（价值） 0 0 0 2 2 2

0 1 1 2 3 3

0 0 0 0 0 0

遍历到i = 5的时候，我们发现，我们可以选择丢掉物品1或2，加入物品3，也可以不加入物品3，如何判断这么多选择呢？我们可以将问题转化为：是 “拥有物品3”还是“不拥有物品3”最优，而当前“不拥有物品3”的最优情况显然是这一列的上一行的数值，对于“拥有物品3”的情况，也即“容量-物品3重量”时的最优价值+物品3价值，我们可以通过查询上一行的对应列得到前者的数值，比较大小后可得出最优情况。如上面的例子，我们对物品3，选择不放入时，dp[4][4] = dp[3][4] = 3，选择放入时，dp[4][4] = dp[3][i - 物品3重量] + 物品3价值 = dp[3][3] + 3 = 5 > 3，我们得到了答案，并且对于任意的物品，我们都可以使用这个二分，得到最优解。写为代码形式，我们有：

dp[i][j] = max(dp[i-1][j-w(i)] + v(i), dp[i-1][j])

自然的，我们可以发现，我们在每一行的赋值过程中，只需要查阅上一行的数据，那么我们可以只保留两行列表，定义列表pre和cur（previous，current）在cur完成一行数据填写后复制给pre，再根据pre填写下一行cur的数值。写为代码形式，我们有：

cur[j] = max(cur[j-w(i)] + v(i), cur[j])

进一步，我们可以发现，在使用上一行列表时只需要搜索容量比当前容量小的列的数值。那么我们可以只保留一列，遍历时从右往左遍历，左边相当于pre部分。代码为：

dp[j] = max(dp[j-w(i)] + v(i), dp[j])

这个时候就体现出从右往左遍历的必要性了，如果从左往右遍历，pre部分会先被修改后再进行应用，见如下的例子，我们让背包容量为5，有物品1和物品2 的价值和重量分别为3/2，2/1。

索引（容量） 0 1 2 3 4 5

值（价值） 0 0 0 3 3 3

对于第二次遍历，我们从左往右遍历到i = 1

索引（容量） 0 1 2 3 4 5

值（价值） 0 2 0 3 3 3

显然dp[1] = 2，那么接下来，dp[2]等于多少呢，我们按dp[j] = max(dp[j-w(i)] + v(i), dp[j])会发现dp[2] = dp[1] + 2 = 4，而这是把物品2重复放入背包得到的，显然不合01背包的题设条件（背包要么有这个物品要么没有）
